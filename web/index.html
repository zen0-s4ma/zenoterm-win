Administracion

Gestión centralizada de usuarios, perfiles, contraseñas y visibilidad del Explorer.

Usuarios del sistema

Perfiles disponibles: administrador, usuario total, usuario sin Docker, usuario para remoto y usuario solo Docker.

Docker Management

Vista centralizada del engine Docker del host, contenedores activos y agrupaciones reutilizables con orden de arranque.

Inventario de contenedores

Crear / editar grupo

Grupos guardados

Lanzamiento secuencial con orden, pull, force-recreate y exportación a script.

Scheduler

Planifica tareas por terminal o por sesión, organiza por categorías y sigue la salida en tiempo real por pestañas.

Crear / editar tarea

Tareas programadas
